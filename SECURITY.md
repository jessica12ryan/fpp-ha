# Security Policy

## Supported Versions

Only the latest version of FPP-HA is supported with security updates.

| Version | Supported |
| ------- | --------- |
| master  | ✅ Yes    |

## Reporting a Vulnerability

If you discover a security vulnerability in FPP-HA, please report it privately by opening a security advisory at:

https://github.com/jessica12ryan/fpp-ha/security/advisories/new

Please do **not** disclose the vulnerability publicly until we have had a chance to address it.

We will acknowledge receipt within 48 hours and provide an estimated timeline for a fix.

## Scope

This security policy covers the FPP-HA Home Assistant add-on repository only. Vulnerabilities in upstream Falcon Player (FPP) should be reported to the [FalconChristmas/fpp](https://github.com/FalconChristmas/fpp) project.
