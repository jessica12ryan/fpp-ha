# FPP - Falcon Player App for Home Assistant

[![FPP logo](https://raw.githubusercontent.com/jessica12ryan/fpp-ha/master/fpp-ha/logo.png)](https://github.com/FalconChristmas/fpp/)

## Getting Started

- After clicking install, it may take a few minutes to download and install FPP. If you refresh or browse to another page, the app will continue downloading. The app page will appear in your app list when the app has been successfully installed.
- You can open FPP by browsing to http://HA_IP
